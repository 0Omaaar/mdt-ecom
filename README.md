## About The Project

This Laravel e-commerce platform provides seamless integration with Dolibarr ERP, enabling real-time product synchronization, inventory management, and order processing between your online store and ERP system.

### Key Features

* **Bi-directional Product Sync**: Automatically sync products between Dolibarr and e-commerce store
* **Real-time Inventory**: Keep stock levels updated across both systems
* **Customer Data Sync**: Maintain consistent customer records
* **Multi-store Support**: Manage multiple storefronts from one admin panel
* **Modern Checkout**: Optimized conversion-focused checkout process
